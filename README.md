## 项目简介

**本项目实现**：

- 网易云音乐爬虫，包括：
  -  用户、排行榜、歌单、歌曲、评论
- 网易云音乐报告生成，包括：
  - 听歌记录、类型分布、歌单喜好
  - 用户评论信息
  - (开发中...)
- 网易云音乐歌曲推荐，包括
  - 基于用户听歌排行榜的协同过滤算法
  - 基于歌曲评论关键字的协同过滤算法
  - (开发中...)
